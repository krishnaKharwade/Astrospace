# Astrospace
ASTROSPACE (SMIS-R) is an AI-based web app for intelligent material selection in space missions. It analyzes conditions like temperature, radiation, and oxygen exposure using machine learning to recommend optimal materials, visualize results, and provide insights such as clustering, feature importance, and Pareto optimization.
